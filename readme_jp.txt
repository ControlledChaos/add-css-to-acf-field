Advanced Custom Fieldsの各フィールドにCSSを追加できるようにするAdd-Onです。

== Description ==

このプラグインを使用すると、Advanced Custom Fieldsの各フィールド毎にCSSを追加して表示を調整することができます。
CSSを追加することで、不要なタイトルを消したり、入力欄の表示を調整して、より使いやすくすることが可能です。
また、独自のフィルターフックを使って、CSSを外部から変更することが可能です。

= Features =

* ACFのフィールド毎にオリジナルのCSS追加が可能。
* ACF Blockにも有効。
* CSSをフィルターフックを使って変更可能。

= Notice =

Advanced Custom Fieldsが必要です。

== Installation ==

1. プラグインをpluginsディレクトリにアップロードします。
2. ダッシュボードにログインし、プラグインを有効化します。
